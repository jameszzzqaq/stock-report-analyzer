## 否决门机制（防止浪费 TOKEN）

框架内置多级否决门，以下任一触发即停止后续分析：

1. **因子1A 否决**：6 项快筛任一不通过 → 停止全部后续
2. **因子1B 否决**：管理层判定"损害价值" → 停止
3. **因子2 分配能力否决**：自由现金流长期为负 → 停止
4. **因子2 粗算否决门**：R < Rf 或 R < II×0.5 → 停止（不进入因子3）
5. **因子4 排除**：安全边际不足 + 价值陷阱风险高 → 排除
